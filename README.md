# Sample Application for Anicya
![snapshot](https://user-images.githubusercontent.com/6835793/55947362-463a1f80-5c89-11e9-813a-988e21b1f5af.png)
## インストール方法
1. Python3 をインストール
2. `./install.sh` を実行

## 実行方法
1. `./start_app.sh` を実行
2. Web ブラウザから http://127.0.0.1:8000/ にアクセス

## メンテナンス
### DB(SQLite) への接続
`./connect_db.sh` を実行

### DB(SQLite) の初期化
`./initialize.sh` を実行
